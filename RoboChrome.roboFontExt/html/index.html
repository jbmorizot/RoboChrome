RoboChrome

Overview

RoboChrome is a RoboFont extension that allows to edit and generate fonts with colored glyphs.

There are different proposals for how colored glyphs can be stored in a font. RoboChrome’s model is based on the Microsoft proposal, which allows a black-and-white glyph to be replaced by a sequence of layered colorized glyphs. That means every layer corresponds to one color. Gradients are not possible. The fonts remain small since the space requirement of a colored glyph is about the same of a normal black-and-white glyph.

The layered glyphs are rasterized to PNG images when you choose to export a font in the Apple color font format. These fonts are slow to generate and will take up considerably more space.

The Adobe/Mozilla proposal allows SVG documents to be stored as glyphs in a font. In RoboChrome, the SVG document is always derived from the sequence of glyph layers, though native SVGs allow much more flexibility, e. g. gradients, animation, and bitmap images.

The bitmap-based Google proposal is not supported in RoboChrome at the moment.
